### Exercise 02: Installing many libraries

В следующих упражнениях вы будете работать с несколькими различными библиотеками. В этом упражнении вам нужно подготовить для них свою виртуальную среду. Установите последнюю версию BeautifulSoup и PyTest. Запрещается устанавливать их по одной (pip install x, pip install y). Запрещается использовать циклы. Найдите умный способ сделать это, используйте установку через требования.

Напишите скрипт python под названием librarian.py, который:

* проверяет, что он запущен в правильной среде
* устанавливает библиотеки
* отображает все установленные библиотеки в конце, как показано ниже (не обязательно, чтобы это был точно такой же список):

    ```
    six==1.14.0
    soupsieve==2.0
    termgraph==0.2.0
    wcwidth==0.1.9
    zipp==3.1.0
    ```
* сохранить его в requirements.txt

Поместите архив вашего env в папку. Вы можете поместить архивацию в свой код или вы можете сделать это из командной строки. Архив может быть сжат, если вы считаете, что это будет полезно. Если скрипт был вызван из неправильного env, должно быть исключение.
